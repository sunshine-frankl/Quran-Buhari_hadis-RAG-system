# Quran-Buhari_Hadith-RAG-System(still doing)

**A Retrieval-Augmented Generation (RAG) system for the Quran and Sahih al-Bukhari Hadith.**

This project aims to build an accurate, trustworthy AI-powered question-answering system grounded in the two primary sources of Islamic knowledge:  
- **The Holy Quran**  
- **Sahih al-Bukhari** (the most authentic collection of Hadith)

---

## 🎯 Project Goal

Create a RAG-based system that allows users to ask questions about Islam and receive answers directly sourced from the Quran and authentic Hadith of Imam Bukhari, minimizing hallucinations and ensuring religious accuracy.

---

## ✨ Features (Planned / In Progress)

- Semantic search over Quran verses and Sahih al-Bukhari Hadith
- Retrieval-Augmented Generation using powerful LLMs
- Support for both Arabic and English (or other languages) queries
- Citation of exact sources (Surah:Ayah or Hadith number)
- Faithfulness & verification mechanisms
- User-friendly interface (Streamlit / Gradio / Web UI)

---

## 🛠️ Tech Stack (Current / Planned)

- **Embedding Models**: Sentence-Transformers, Arabic-specific models, or multilingual
- **Vector Database**: Chroma, FAISS, Pinecone, or Qdrant
- **LLM**: Open-source models (Llama 3, Mistral, Qwen, Gemma) or Groq / OpenAI
- **Framework**: LangChain / LlamaIndex
- **Frontend**: Streamlit or Gradio
- **Data Processing**: Quran JSON + Sahih Bukhari dataset
